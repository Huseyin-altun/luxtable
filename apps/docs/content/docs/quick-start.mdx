## Method 2: createColumnsFromData - Auto from JSON

**Fastest method!** Provide your JSON data, columns are generated automatically. Only specify where you want custom rendering:

<LuxTableAutoDemo />

```tsx
"use client";

import { LuxTable, createColumnsFromData, StatusCell, ProgressCell } from "luxtable";

const data = [
  { firstName: "Tanner", lastName: "Linsley", age: 24, status: "Active", progress: 50 },
  { firstName: "Jane", lastName: "Doe", age: 32, status: "Pending", progress: 80 },
];

// ðŸš€ Columns ready in one line!
const columns = createColumnsFromData(data, {
  // English headers (optional - camelCase is automatically converted to Title Case)
  headers: {
    firstName: "First Name",
    lastName: "Last Name",
    age: "Age",
    status: "Status",
    progress: "Progress",
  },
  // Only specify columns requiring custom appearance
  cells: {
    status: (info) => <StatusCell value={info.getValue() as string} />,
    progress: (info) => <ProgressCell value={info.getValue() as number} showLabel />,
  },
  // Hide unwanted columns (optional)
  // exclude: ["id", "createdAt"],
});

export function MyTable() {
  return <LuxTable data={data} columns={columns} />;
}
```

  ## Built-in Cell Components

LuxTable comes with ready-made components:

| Component | Description | Props |
|---|---|---|
| `StatusCell` | Colored badge | `value`, `colors?`, `className?` |
| `ProgressCell` | Progress bar | `value`, `barColor?`, `bgColor?`, `showLabel?` |
| `BooleanCell` | Yes/No display | `value`, `trueLabel?`, `falseLabel?` |
| `DateCell` | Date formatting | `value`, `format?` (`short`/`long`/`relative`) |
| `CurrencyCell` | Currency | `value`, `currency?`, `locale?` |

### Examples

```tsx
// StatusCell - Default colors: Active, Inactive, Pending, Completed, Cancelled
<StatusCell value="Active" />

// ProgressCell - Progress bar with label
<ProgressCell value={75} showLabel />

// BooleanCell - Custom labels
<BooleanCell value={true} trueLabel="Yes" falseLabel="No" />

// DateCell - Relative date
<DateCell value={new Date()} format="relative" />

// CurrencyCell - Turkish Lira
<CurrencyCell value={1500} currency="TRY" locale="tr-TR" />
```
