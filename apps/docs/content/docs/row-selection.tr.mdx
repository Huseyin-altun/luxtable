# Satır Seçimi

LuxTable, yerleşik checkbox işlevselliği ile hem tekli hem de çoklu satır seçimini destekler.

## Canlı Demo

<LuxTableSelectionDemo />

## Temel Kullanım

Seçim modunu belirterek satır seçimini etkinleştirin:

```tsx
<LuxTable
  data={data}
  columns={columns}
  options={{
    selection: "multiple", // veya "single"
  }}
  onSelectedRowsChange={(selectedRows) => {
    console.log("Seçilen:", selectedRows);
  }}
/>
```

---

## Seçim Modları

| Mod | Açıklama |
|-----|----------|
| `"none"` | Seçim yok (varsayılan) |
| `"single"` | Aynı anda bir satır seç |
| `"multiple"` | Checkbox ile birden fazla satır seç |

---

## Seçili Verileri Alma

Seçili satır verilerini almak için `onSelectedRowsChange` callback'ini kullanın:

```tsx
<LuxTable
  data={data}
  columns={columns}
  options={{ selection: "multiple" }}
  onSelectedRowsChange={(selectedRows) => {
    // selectedRows seçili satırların tam veri nesnelerini içerir
    console.log("Seçili kayıtlar:", selectedRows);
    
    // Örnek: Seçili satırların ID'lerini al
    const selectedIds = selectedRows.map(row => row.id);
  }}
/>
```

## Kontrollü Seçim

İleri düzey kullanım durumları için seçim durumunu dışarıdan kontrol edebilirsiniz:

```tsx
import { useState } from "react";
import { LuxTable, RowSelectionState } from "luxtable";

export default function App() {
  const [rowSelection, setRowSelection] = useState<RowSelectionState>({});

  return (
    <>
      <LuxTable
        data={data}
        columns={columns}
        rowSelection={rowSelection}
        onRowSelectionChange={setRowSelection}
        options={{ selection: "multiple" }}
      />
      
      <div>
        Seçili sayısı: {Object.keys(rowSelection).length}
      </div>
    </>
  );
}
```

### RowSelectionState Formatı

Seçim durumu, anahtarların satır indeksleri olduğu bir nesnedir:

```tsx
// Hiç satır seçili değil
const rowSelection = {};

// Birinci ve üçüncü satırlar seçili
const rowSelection = { "0": true, "2": true };
```

---

## Seçim Checkbox'ını Gizleme

Görünür checkbox olmadan seçim istiyorsanız (örn. satır tıklama seçimi için):

```tsx
<LuxTable
  data={data}
  columns={columns}
  options={{
    selection: "single",
    showSelectionCheckbox: false,
  }}
/>
```

## Toplu İşlem Örneği

Satırlar seçildiğinde toplu işlem butonları göstermek yaygın bir desendir:

```tsx
import { useState } from "react";
import { LuxTable, RowSelectionState } from "luxtable";

export default function App() {
  const [rowSelection, setRowSelection] = useState<RowSelectionState>({});
  
  const selectedCount = Object.keys(rowSelection).length;

  const handleBulkDelete = () => {
    // Seçili satır indekslerini al
    const selectedIndices = Object.keys(rowSelection).map(Number);
    console.log("Silinen satırlar:", selectedIndices);
    // Silme işlemini gerçekleştir...
    setRowSelection({}); // Seçimi temizle
  };

  return (
    <div>
      {selectedCount > 0 && (
        <div className="mb-4 p-3 bg-blue-50 rounded flex items-center gap-4">
          <span>{selectedCount} satır seçili</span>
          <button onClick={handleBulkDelete} className="text-red-600">
            Seçilenleri Sil
          </button>
        </div>
      )}
      
      <LuxTable
        data={data}
        columns={columns}
        rowSelection={rowSelection}
        onRowSelectionChange={setRowSelection}
        options={{ selection: "multiple" }}
      />
    </div>
  );
}
```
